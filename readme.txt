1.C compiler should be installed on your computer.

You can check whether it is installed or not by typing and entering "gcc --version" command in command promt or terminal (works in windows/linux/mac). If it returns some text including a version number, gcc compiler has installed on your computer. If not, you install it by following the instructions provided below.

windos  :   (https://www.scaler.com/topics/c/c-compiler-for-windows/)
linux   :   (https://linuxize.com/post/how-to-install-gcc-compiler-on-ubuntu-18-04/)
macOS   :   (https://www.scaler.com/topics/c/c-compiler-for-mac/)


2.compile and run 

open containing folder in terminal/command promt. enter "gcc stringmatcher.c -o stringmatcher" command.
then execute "stringmatcher".
